using ara::crypto::AllowedUsageFlags = typedef std::uint32_t

‪A container type and constant bit-flags of allowed usages of a key or a secret seed object.

Only directly specified usages of a key are allowed, all other are prohibited! Similar set of flags are defined for the usage restrictions of original key/seed and for a symmetric key or seed that potentially can be derived from the original one.
